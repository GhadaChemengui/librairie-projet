# 📚 Système de Gestion de Librairie

Une application complète de gestion de librairie développée avec Laravel, Docker et MySQL.

## 🚀 Fonctionnalités

- Gestion des livres et auteurs
- Système d'authentification avec rôles
- Interface d'administration
- Base de données MySQL
- Conteneurisation Docker

## 🛠️ Stack Technique

- **Backend**: Laravel 10
- **Frontend**: Laravel Blade + Tailwind CSS
- **Base de données**: MySQL 8.0
- **Serveur Web**: Apache
- **Conteneurisation**: Docker & Docker Compose

## 📦 Prérequis

- Docker
- Docker Compose

## 🏃‍♂️ Installation

\\\ash
docker-compose up -d
docker-compose exec frontend php artisan migrate
